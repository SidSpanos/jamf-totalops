# Create Profiles – Details

This document outlines the structure and best practices when creating configuration profiles in JAMF Pro.

---

## 🔐 Security Payloads
- Password policy enforcement
- FileVault settings
- Gatekeeper and Privacy preferences
- Jamf Protect configuration profile inclusion

## 🌐 Network Payloads
- Wi-Fi auto-connect and fallback setup
- VPN on-demand settings
- Proxy auto-configuration (PAC) URLs

## 🧩 App Management
- Web clips and Home screen layout for iOS
- Managed app configuration keys
- Pre-approval for System Extensions

## 📈 Restrictions
- Block app installation/removal
- Disable camera, AirDrop, or screenshots
- Block gaming, social media, or streaming apps

## 🧪 Best Practices
- Use meaningful profile names
- Separate payload types into different profiles
- Test before broad deployment
- Monitor logs after profile deployment

# DEP Config – Details

This document describes how to properly configure DEP (Device Enrollment Program) in Apple Business Manager for JAMF.

---

## 🧭 Device Enrollment Settings
- Assign MDM server (JAMF) to devices
- Ensure supervision is enabled
- Allow pairing with other devices (optional)
- Configure default profile to skip setup steps

## 📋 Setup Assistant Customization
- Skip Siri, Apple ID, Terms & Conditions, etc.
- Customize department and contact fields

## 🔐 MDM Authentication
- Optional user auth during setup
- Use SSO or JAMF Connect integrations

## 🧪 Testing
- Assign one device to a test profile
- Factory wipe to trigger new DEP flow
- Use Apple Configurator 2 for manual assignment
