# DEP Config – Details

This document describes how to properly configure DEP (Device Enrollment Program) in Apple Business Manager for JAMF.

---

## 🧭 Device Enrollment Settings
- Assign MDM server (JAMF) to devices
- Ensure supervision is enabled
- Allow pairing with other devices (optional)
- Configure default profile to skip setup steps

## 📋 Setup Assistant Customization
- Skip Siri, Apple ID, Terms & Conditions, etc.
- Customize department and contact fields

## 🔐 MDM Authentication
- Optional user auth during setup
- Use SSO or JAMF Connect integrations

## 🧪 Testing
- Assign one device to a test profile
- Factory wipe to trigger new DEP flow
- Use Apple Configurator 2 for manual assignment
